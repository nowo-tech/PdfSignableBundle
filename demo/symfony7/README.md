# PdfSignable Bundle – Demo (Symfony 7)

Demo app for **nowo-tech/pdf-signable-bundle** on Symfony 7. Runs with FrankenPHP over HTTPS on localhost.

## Quick start

From the bundle root (recommended):

```bash
make run-symfony7   # http://localhost:8001
```

From this directory:

```bash
make run
```

Accept the self-signed certificate in the browser. See the [parent README](../README.md) for Xdebug, Web Profiler, and assets.

## Makefile

Run `make help` for available targets (e.g. `up`, `down`, `install`, `update-bundle`).
